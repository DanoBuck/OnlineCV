Summary of Features

This website consists of:

  • HTML5
  • CSS
  • Twitter Bootstrap


HTML5

The HTML5 is basic tags which allow for the structure of the website.


CSS

The CSS has been modified to help the website to look the way it does. This is used by changed default options provided through the Twitter bootstrap CDN.


Twitter Bootstrap

The Twitter Bootstrap is the bones of the CSS for this website. It provides a lot of class names which I have plugged into my website to give it the bootstrap look and feel.

Features used of Twitter Bootstrap

I have used a lot of features in the Twitter Bootstrap collection. These include:

  • Navbar - for navigating around the website
  • data-spy's - which link up to targets such as navbars and highlight different sections on the website
  • Collapsible navbars - which take effect when the screen size decreases - it looks well on mobile and small devices
  • Grids - allows me to break a screen into different sections by using Twitter Bootstrap grids
  • Buttons - appealing button design
  • Responsive images - allowing images to take different shapes on the website
  • Modals - used for the contact form, work experience, and projects. Makes it interactive for the users as they choose what the went to see more of
  • Typography - to make the text look nice
  • Collapsible sections - the user can click buttons which makes something appear
  • Pills/Tabs - which can be used to hide or see the selected information
  • Tables - to show data in a neat readable format
  • Footer


Dynamic Aspects

I have implemented some dynamic features to this website. These include using:

  • AngularJS
  • JQuery
  • JavaScript
  • Enformed
  • Downloadable C.V.
  • Football API
  • Weather API
  • System Clock


AngularJS

Angular is used in this site to populate a table in the education section. To do this I created a new JavaScript file and created a controller. In this controller I created a JSON array which I then inserted my college results into. When this was complete, I called this array in my website. Therefore, populating the table dynamically.


JQuery

JQuery is used in this site to validate the contact form. This ensures that the user enters in a valid, in syntax, email address and that a message and name are entered. A recaptcha is also used on the form to stop spam bots from spamming me. JQuery is also used for smooth scrolling with the tabs. A user can click on a tab in the navbar and they will be brought to the corresponding section via JQuery. Another nice interactive feature.


JavaScript

JavaScript is used to connect to the GoogleMaps API and display where I am located to the user.


Enformed

Enformed is a free service which allows you to send the data that has been entered in your form to your email address.


Downloadable C.V.

A link which allows the user to download and look at my latest C.V.

Football API

I have implemented a dynamic component to the website for Latest Liverpool F.C. fixtures i.e. their next match taking place. I used JavaScript and AJAX calls to api.football-data.org. This took a bit of work to do but was quite enjoyable manipulating the JSON response from the API. The script is located in the js folder in the root of the structure. When the API is updated by the owner, the next football match for Liverpool, as I have implemented logic to work this out, will appear. I am very happy about how this works. I thought of this functionality as I constantly ask people who Liverpool are playing. Now I will not need to. The date and time of the match has been formatted through my created functions.


Weather API

I have given the website the ability of displaying the current weather for Tallght, Dublin. This is done by making another JavaScript and AJAX call to a weather API which is located at https://api.forecast.io/forecast/. I send a GET request to this API to get the JSON response back from it. When I recieve a response I need to do more processing to it. The degrees I receive back is in Fahrenheit which most Irish people don't care about. This is not the response I wanted so I have implemented a function to convert this metric to Celcius. The date was also returned in the American format instead of the Irish format. I also wrote a function to format this the correct way.


System Clock

Created a clock in JavScript which shows the user the time in real-time. When the website is open the clock will constantly be changing. This shows the user the time in real-time when they are on the website. The clock is located at the bottom of the screen in the current weather section.
